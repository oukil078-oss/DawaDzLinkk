## 🎥 Option bonus : Vidéo de démonstration

Pour un meilleur impact, enregistrez une vidéo de 3-5 minutes montrant :
1. La page d'accueil
2. Connexion en tant que Pharmacie → Parcourir les produits → Ajouter au panier
3. Connexion en tant que Fournisseur → Voir les commandes → Confirmer une commande
4. L'import Excel avec l'IA

Outils gratuits pour enregistrer : Loom, OBS Studio, ou simplement l'enregistrement d'écran Windows/Mac.
